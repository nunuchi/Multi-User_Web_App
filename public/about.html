Phaser V3

Primarily used Phaser V3 to develope the game. As V3.0 came out recently, in February, coding limitations and documentations were limited.

Express.js

Express, is a web app framework that we used with Node.js to build this app. It's needed for the other modules. Express allows for us to do GET/POST with web pages, thus allowing us to transfer to the ejs pages with query strings.

Socket.io

Allows for us to setup the socket for the server and also allows for us to emit and transfer data from the game to all players. So we would use this to get information of where the players/jewels/obstacles are and place them accordingly in other player's screen.

BodyParser.js + Query-String

BodyParser is a middleware for Express, it is used to handle HTTP POST requests. It gets the requests, with 'req', and then we can use it in our codes. We got the query-string from the html link and used it to display the username and their color while also using it to setup their team based on the color.

PATH

Allows for file and directory path usage. Sends the users to them.

Validator

With this module, we used Validation to check the input values are valid. We primarily used this module for Sanitization, in that we removed characters and such from the input data.

HTTP/HTTPS

Allows for us to setup the HTTP web server and client.
